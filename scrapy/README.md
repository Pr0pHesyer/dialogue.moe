# scrapy

## Install

with python3

```bash
pip install scrapy
```

## Start

```bash
mkdir data_subject
mkdir data_sub
scrapy crawl crawl_subject
scrapy crawl crawl_sub
```

Or you can download the crawled data from [Google Drive](https://drive.google.com/open?id=1yrzv7NC9Bvf9Mk4x9bbYQ54b_bLWqlDJ) (updated on 2019/6/27)
